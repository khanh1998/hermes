# hermes
An instant messaging app that delivers your messages as fast as Hermes

# Develop

1. sudo docker-compose up -d

Wait about 2 minutes, kafka takes time to get ready

2. cd client
npm run dev

3. cd auth
npm run start:dev

4. cd api
npm run start:dev

5. cd shipping
go run .

6. cd socket
go run .

go to localhost:3000
